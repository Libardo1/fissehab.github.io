R_netcdf

Correlation Map from NetCDF

In this first example, we display how to use R to analyse climate data. Most climate data are in NetCDF format which is an array that contains different variables and dimensions. Now, let use see how to open netcdf data in R and generate correlation map between global sea surface temperature and rainfall over xxx in the yyy season. First we will import the required libraries, then we will open the monthly average global temperature data
